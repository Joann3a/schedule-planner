# 日程规划助手

一个集 **AI 目标拆解**、**日历视图**、**番茄钟** 于一体的个人效率工具，纯前端实现，数据本地存储。

## 功能

- **AI 目标规划** — 输入目标（如"一个月学完高数"），调用 AI 自动拆解为带时间节点的任务清单
- **日历视图** — 按天/周展示任务，点击标记完成
- **番茄钟** — 选中任务后启动自定义时长的专注计时，自动记录专注次数
- **数据本地存储** — 所有数据存在浏览器 localStorage，无需后端

## 使用

1. 打开 https://joann3a.github.io/schedule-planner/
2. 点击右上角 ⚙️ 配置 API Key（支持 OpenAI 兼容接口）
3. 在「AI 目标规划」页输入目标，点击 AI 智能拆解
4. 保存后在日历中查看任务，用番茄钟专注执行

### 移动端

在 iPhone Safari 打开后，分享 → 添加到主屏幕，即可像原生 App 一样全屏使用。

## 技术

纯静态 HTML 页面，无框架无构建工具：

- 原生 HTML/CSS/JS
- CSS Grid + 响应式布局（适配桌面和移动端）
- SVG 圆环进度动画
- localStorage 数据持久化
- OpenAI 兼容 API 集成
- PWA / iOS Web App 支持

## 本地运行

```bash
python serve.py
```

然后在同一 WiFi 下的手机上访问 `http://你的电脑IP:8080`。
